# Sei IDE

A custom web IDE for the Sei programming language.

## ✨ Features

- **Full Sei IDE**: Write, run, and debug Sei code in your browser
- **Interactive Terminal**: Real-time output and input handling
- **Quick Reference**: Built-in syntax reference cards
- **Example Programs**: Load pre-built examples to learn Sei

## 🚀 Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Open http://localhost:5173 in your browser

## 📚 Sei Language

Sei is a programming language with Tanglish syntax:
- `Vai` - Declare variables
- `Kaami` - Print output
- `Sei` - Define functions
- `Ippo/Illana` - If/Else conditions
- `Varai` - While loops
- And much more!

View full documentation in the IDE or check the Documentation tab.
